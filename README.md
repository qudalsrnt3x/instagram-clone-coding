 🔑 기능 설명
---
### 1) 회원가입/로그인

- JWT인증 방식으로 로그인 구현
- ID 중복확인, 각 필드별 유효성 체크

### 2) 게시글 CRUD

- 게시글 목록 전체 조회, 상세조회, 등록, 삭제
- 이미지 업로드, 삭제, 수정

### 3) 댓글 CRUD

- 댓글 등록, 조회, 삭제
